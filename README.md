# smartphone-usage-dashboard
An interactive Excel dashboard analyzing smartphone usage and addiction patterns across 3 demographic dimensions: gender, age group, and usage category. Built using Pivot Tables, Slicers, and advanced charts to enable dynamic filtering and multi-dimensional exploration.
# Dataset
Source: Kaggle 
# Key Findings
 • Young professionals showed the highest average daily usage (~7.53 hrs), exceeding adults and young adults
 • Weekend screen time follows a similar gender distribution to weekday usage, suggesting consistent behavioral patterns
 • The majority of users fall into Mild or Moderate addiction levels, with Severe representing a smaller but notable segment
# Tools Used
Microsoft Excel — Pivot Tables, Slicers, Bar Charts, Donut Charts, Dashboard Layout
